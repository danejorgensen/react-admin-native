# react-admin-native

> [react-admin](https://github.com/marmelab/react-admin) files needed for react native

## Install

```bash
$ npm install --save @npmdanej/react-admin-native
```

### View in browser

```bash
$ cd example
$ npm start
```

### Test

```bash
$ npm run test
```

### Deploy to NPM

1. Update version in package.json
2. Publish to NPM

```bash
$ npm publish
```
